# 情绪音乐盒

> AI-SDD 5 阶段工作流项目

## 简介
情绪音乐盒是一个把用户当前心情转化为音乐参数、动态可视化和情绪音乐卡的初赛交互 Demo。

## 技术栈
Vite + React + TypeScript + localStorage + Web Audio API

## 快速开始
```bash
cd frontend
npm install
npm run dev
```
